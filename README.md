# 🥀 THE DIGITAL GRIMOIRE
`STATUS: ONLINE`

An interactive terminal interface developed as a personal archive, merging competitive programming performance with a Victorian/Gothic aesthetic. This project functions as a digital grimoire, serving as both a professional portfolio and a hub for academic tools.

## 🔮 01. THE ALMANAC
The system utilizes JavaScript logic to create a dynamic, contextual environment:
*   **Specimen of the Day**: A botanical catalog that rotates through 73 historical etchings. The system processes the scientific Latin names of the plants to retrieve the corresponding `.png` file from the `assets/specimens/` directory.
*   **Moon Phase Logic**: An algorithm that calculates the approximate lunar phase based on Unix time, displaying poetic descriptions of the moon's current illumination.
*   **Real-time Context**: Integration with `ip-api` and `open-meteo` APIs to display live location and current temperature data.

## 🛠️ 02. GRIMOIRE SKILLS (TECH STACK)
Software engineering focused on clean front-end logic and high-contrast UI:
*   **Visual Engineering**: Extensive use of CSS filters and `mix-blend-mode` to replicate the texture of ink on aged paper.
*   **Responsiveness**: Dynamic text input (`textarea`) with JS-driven auto-expansion to accommodate long-form messages.
